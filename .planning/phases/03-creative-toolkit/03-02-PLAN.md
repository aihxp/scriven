**Create theme-tracker.md** (STRUCT-03, implements D-08):
- YAML frontmatter: `description: Track thematic threads across the work with auto-detection suggestions.`
- Title: `/scr:theme-tracker -- Track Thematic Threads`
- Instruction loads: config.json, CONSTRAINTS.json (file_adaptations), THEMES.md, adapted characters file, drafted prose (if any)
- DISPLAY MODE: Show all tracked themes from THEMES.md with which units they appear in, strength indicators, and supporting quotes
- AUTO-DETECT MODE: Scan drafted prose for recurring motifs, symbols, and thematic patterns. Present detected themes as SUGGESTIONS with evidence quotes. Per D-08: NEVER auto-add to THEMES.md. Always ask "Add this theme? [y/n]" for each suggestion
