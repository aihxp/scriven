    1. **Temp environment isolation.** Each test creates its own tmpdir via `fs.mkdtempSync`. Cleanup via `fs.rmSync(dir, { recursive: true, force: true })` in a try/finally or `after()` hook. Do NOT touch the real `~/.scriven` or `~/.claude` or `~/.codex` directories.
