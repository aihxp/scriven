Create two quality review commands following the established command pattern.

**dialogue-audit.md** (QUAL-03):
- YAML frontmatter: `description: Audit dialogue for character voice differentiation, attribution clarity, and talking-head detection.`
- Usage: `/scr:dialogue-audit [N]`
- Instruction: Role is "dialogue specialist". Load config.json, CONSTRAINTS.json (this command is hidden from academic, poetry, speech_song, sacred per CONSTRAINTS.json), CHARACTERS.md (for voice anchors), and drafted prose
- Check categories:
  1. Voice differentiation -- Can you tell who's speaking without dialogue tags? Each character should have distinct vocabulary, sentence length, verbal tics, and speech patterns compared to their voice anchor in CHARACTERS.md
  2. Attribution clarity -- Is it always clear who's speaking? Flag ambiguous stretches of unattributed dialogue (more than 3 exchanges without a tag or action beat)
  3. Dialect consistency -- If a character uses dialect, is it consistent throughout? Flag any dropped or inconsistent dialect features
  4. Talking-head detection -- Flag scenes where characters talk without physical action, setting interaction, or internal thought. Pure dialogue exchanges lasting more than half a page should be flagged
- Output: For each issue show the passage, the characters involved, the category, and a specific suggestion
- Save to `.manuscript/{scope}-DIALOGUE-AUDIT.md`

**pacing-analysis.md** (QUAL-04):
- YAML frontmatter: `description: Generate a structure-aware pacing report analyzing scene tempo and narrative flow.`
- Usage: `/scr:pacing-analysis [N]`
- Instruction: Role is "pacing analyst". Load config.json, CONSTRAINTS.json (hidden from poetry/speech_song), OUTLINE.md (for structural context -- which scenes are climaxes, which are breathers), and drafted prose
- Analysis dimensions:
  1. Scene length distribution -- word counts per scene/chapter, identify outliers
  2. Tempo mapping -- classify each scene as action/tension/revelation/transition/reflection, map the resulting rhythm across the narrative
  3. Climax vs breather balance -- are high-tension scenes followed by breathers? Are there too many consecutive scenes at the same intensity?
  4. Opening and closing energy -- does each chapter/scene open with enough hook? Do endings propel the reader forward?
  5. Saggy middle detection -- flag the 40-60% mark if pacing drops without intentional purpose
- Output: Visual tempo map using block characters or similar text visualization, plus specific findings per dimension
- Save to `.manuscript/{scope}-PACING-REPORT.md`

Create the sensitivity-review command (QUAL-06).

**sensitivity-review.md**:
- YAML frontmatter: `description: Flag potential sensitivity issues with context, suggest alternatives, and note intentional craft.`
- Usage: `/scr:sensitivity-review [N]`
- Instruction: Role is "sensitivity reader". Load config.json, CONSTRAINTS.json (adapted: ethics-review for academic, interfaith-review for sacred), and drafted prose
- CRITICAL per research pitfall 4: This command must distinguish between unintentional insensitivity and deliberate artistic choices. It must NOT flag everything as problematic. A gritty crime novel should not be flagged for depicting crime.
- Review categories:
  1. Representation -- How are marginalized groups depicted? Are portrayals nuanced or stereotypical?
  2. Cultural accuracy -- Are cultural practices, traditions, and beliefs portrayed accurately?
  3. Language sensitivity -- Slurs, outdated terminology, potentially harmful phrasing -- is usage intentional and contextualized?
  4. Power dynamics -- Are power imbalances depicted thoughtfully? Is trauma handled with care?
  5. Intentional craft recognition -- For each flagged item, assess whether this appears to be intentional artistic choice or unintentional insensitivity. If the genre/tone/character context supports the choice, note "Likely intentional craft" and explain why
- Output format for each finding:
  - Passage and location
  - Concern category
  - Assessment: "Potentially unintentional" or "Likely intentional craft"
  - If unintentional: suggested alternative
  - If intentional: note acknowledging the artistic choice and any considerations for the writer
- Explicitly state in instructions: "You are a thoughtful reader, not a censor. Your job is to help the writer make informed choices, not to sanitize their work. Note where craft choices are deliberate."
- Save to `.manuscript/{scope}-SENSITIVITY-REVIEW.md`
