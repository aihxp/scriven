  <action>
Add four platform package sections to export.md after the format sections.

**kdp-package** (EXP-10, per D-05):
- First run pdf --print-ready export (or verify it already exists in .manuscript/output/)
- Calculate page count: ceil(word_count / 250)
- Calculate spine width per D-05: `spine_width = (page_count * paper_factor) + 0.06`
  - paper_factor: white=0.002252, cream=0.0025, color=0.0032
  - Read paper_type from config.json (default: white)
- Calculate cover dimensions:
  - trim_width from config.json (default: 6in), trim_height (default: 9in)
  - cover_width = (trim_width * 2) + spine_width + (0.125 * 2) for bleed
  - cover_height = trim_height + (0.125 * 2) for bleed
- Spine text rule: only add text to spine if page_count >= 79
- Create .manuscript/output/kdp-package/ directory
- Copy interior PDF, generate cover-specs.md with exact dimensions, generate kdp-metadata.md with title/author/categories/keywords

**ingram-package** (EXP-11):
- Requires print-ready PDF (run or verify exists)
- Check `command -v gs` -- if missing, provide install: `brew install ghostscript` / `apt install ghostscript`
- Convert to CMYK PDF/X-1a: `gs -dPDFX -dBATCH -dNOPAUSE -dPDFXCompatibilityPolicy=1 -sColorConversionStrategy=CMYK -sDEVICE=pdfwrite -sOutputFile=.manuscript/output/ingram-package/manuscript-cmyk.pdf .manuscript/output/manuscript-print.pdf`
- Warn about ICC color profiles -- CMYK conversion quality depends on system ICC profiles
- Create .manuscript/output/ingram-package/ with CMYK PDF + cover specs + metadata

**query-package** (EXP-12):
- Bundle: query letter (.manuscript/output/query-letter.md or generate via /scr:query-letter), synopsis (.manuscript/output/synopsis.md or generate), first 3 chapters from OUTLINE.md
- Check prerequisites exist, offer to generate missing pieces
- Assemble into .manuscript/output/query-package/ directory
- Also create a single combined DOCX: `pandoc query-letter.md synopsis.md sample-chapters.md -o query-package.docx --reference-doc=data/export-templates/scriven-manuscript.docx`

**submission-package** (EXP-13):

- Check prerequisites, offer to generate missing
- Assemble into .manuscript/output/submission-package/ directory
