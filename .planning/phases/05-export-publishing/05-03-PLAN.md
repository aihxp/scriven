  <action>
Create manuscript-stats command following established command pattern (YAML frontmatter, Usage, Instruction sections).

**YAML frontmatter:**
- description: "Show manuscript word count, chapter count, estimated page count, and reading time."

**Usage:** `/scr:manuscript-stats [--detail]`
- `--detail` shows per-unit breakdown (word count per chapter/scene)

**STEP 1: LOAD CONTEXT**
- Load config.json for work type and structural hierarchy names
- Load OUTLINE.md to get total unit count and unit names

**STEP 2: COUNT WORDS**
- Read all files in .manuscript/drafts/body/
- Count words in each file (split on whitespace, exclude markdown formatting markers)
- Track per-unit word counts for --detail mode
- Also count words in .manuscript/front-matter/ and .manuscript/back-matter/ separately

**STEP 3: CALCULATE METRICS**
- Total word count (body only, exclude front/back matter)
- Full word count (body + front + back matter)
- Chapter/unit count (from OUTLINE.md total units)
- Drafted unit count (files that exist in .manuscript/drafts/body/)
- Draft completion: `drafted / total * 100`%
- Estimated page count: `ceil(full_word_count / 250)`
- Reading time (average): `ceil(full_word_count / 250)` minutes, format as Xh Ym
- Reading time (careful): `ceil(full_word_count / 200)` minutes
- Front matter elements: count of files in .manuscript/front-matter/
- Back matter elements: count of files in .manuscript/back-matter/

**STEP 4: DISPLAY**
Show clean formatted output:

```
Manuscript Statistics
====================

Title: {title} by {author}
Work Type: {work_type}

Words:      {total_word_count} (body)
            {full_word_count} (with front/back matter)
Units:      {drafted}/{total} ({completion}% complete)
Pages:      ~{page_count} (estimated at 250 words/page)
Reading:    ~{reading_avg} (average) / ~{reading_careful} (careful)

Front Matter: {front_count} elements
Back Matter:  {back_count} elements
```

If --detail, add per-unit table:
```
Unit Breakdown
--------------
Ch 1: The Beginning     3,200 words   ~13 pages
Ch 2: The Middle        4,100 words   ~16 pages
...
```
