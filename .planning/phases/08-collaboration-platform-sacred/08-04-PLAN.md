  - Brief description

**`genealogy.md` — Build and verify genealogical trees:**
- Reads FIGURES.md and LINEAGES.md
- `--build`: generate genealogical tree from existing figure data and drafted text
- `--verify`: check genealogical consistency (no impossible relationships, dates align, no circular lineage)
- Output to `.manuscript/GENEALOGY.md` with tree notation (indented hierarchy)
- Support multiple lineage types: biological, covenantal, spiritual, royal
- Flag discrepancies with drafted text

**`chronology.md` — Timeline with tradition-appropriate dating:**
- Reads `config.json` for `sacred.calendar_system` (gregorian, hebrew, hijri, vikram_samvat, buddhist_era, regnal, multiple)

- Generates `.manuscript/CHRONOLOGY.md` with events in chronological order using the specified calendar system
- `--calendar <system>`: override calendar system for display
- Support dual dating (e.g., "587 BCE / 3174 AM") when `calendar_system` is `multiple`
- Flag chronological conflicts

**`annotation-layer.md` — Commentary layer alongside primary text:**
- Creates or edits `.manuscript/annotations/{tradition}-annotations.md`
- `--tradition <tradition>`: which commentary tradition (from `sacred.annotation_traditions` config)
- Each annotation links to a specific passage (unit:verse) with commentary text
- Supports multiple layers: same passage can have annotations from different traditions (e.g., Catholic, Reformed, Orthodox)
- Display mode: show primary text with annotations interleaved
- Edit mode: add/edit annotation for a specific passage

**`verse-numbering.md` — Manage numbering systems:**
- Reads `config.json` for `sacred.verse_numbering_system` (masoretic, septuagint, quranic_hafs, quranic_warsh, pali_canon, custom)
- `--apply`: add verse numbers to drafted units based on numbering system
- `--system <system>`: set or switch numbering system
- `--verify`: check that all units have consistent numbering
- `--dual`: show dual numbering (e.g., Masoretic alongside Septuagint where they differ)
- Store numbering metadata in `.manuscript/verse-map.json`

**`source-tracking.md` — Track primary sources, oral traditions, variants:**
- Document sources for each passage: manuscript tradition, textual variants, oral tradition lineage
- `--add <passage> <source>`: record a source for a passage
- `--variants <passage>`: show known textual variants for a passage
- `--report`: generate source analysis report
- Output to `.manuscript/SOURCES.md`
- Useful for critical editions where provenance matters

- Severity levels: `contradiction` (directly opposes doctrine), `tension` (may conflict), `novel` (not addressed in DOCTRINES.md)

**Enhance `agents/drafter.md` with full register definitions (SACRED-02, per D-05):**

The drafter already has a "Sacred/historical work types" section mentioning registers. Expand it with detailed guidance for each of the 10 registers:

Add a "### Voice Register Reference" subsection with a definition block for each register:

**Prophetic:** Urgent, declarative, oracular. Short sentences, imperative mood, "Thus says the Lord" framing. Direct address to audience. Present/future tense dominance. Repetition for emphasis ("Woe, woe, woe"). Exclamatory. No hedging.

**Wisdom:** Aphoristic, reflective, balanced. Parallelism (antithetical, synonymous, synthetic). Comparative structure ("Better X than Y"). Observational tone. Third person often. Conditional constructions ("If you...then"). Measured cadence.

**Legal / Halakhic:** Precise, conditional, imperative. Case-law structure ("If a man does X, then Y"). Categorical. Exhaustive enumeration. No ambiguity tolerated. Present tense. Second person direct address for commandments ("You shall / You shall not").

**Liturgical:** Formal, rhythmic, responsive. Call-and-response patterns. Doxological language (praise, blessing, thanksgiving). Repetitive refrains. Elevated vocabulary. Present tense. Address to the divine or congregation. Musical awareness.

**Narrative-historical:** Chronicle-like, temporal, factual. Sequential narration ("And it came to pass"). Genealogical asides. Geographic precision. Past tense. Third person. Minimal editorializing. Wayyiqtol-style consecutive narration rhythm.

**Apocalyptic:** Visionary, symbolic, cosmic. "I saw..." framing. Numbers as symbols (7, 12, 40). Animal imagery. Throne-room scenes. Cosmic battle. Dream logic. Urgency. Future tense and prophetic perfect. Interpreting angel as guide.

**Epistolary:** Personal, didactic, pastoral. Greeting formula. "Dear brothers/sisters" address. Practical instruction. Theological argument. Exhortation. Closing benediction. Mix of personal warmth and doctrinal authority.

**Poetic / Psalmic:** Musical, metaphorical, parallelism-heavy. Hebrew parallelism patterns. Nature imagery. Emotional range (lament, praise, thanksgiving, royal). Selah markers. Acrostic potential. Chiastic structures. Address to God and self ("O my soul").

**Parabolic:** Story-within-story, allegorical. "The kingdom of heaven is like..." framing. Concrete imagery from daily life (seeds, sheep, merchants). Surprising twist. Open-ended or with explicit moral. Accessible surface, deep meaning.

**Didactic:** Instructional, systematic, expository. Topic-by-topic structure. Definition and explanation. Q&A format possible. Teacher-student dynamic. Building from simple to complex. Summary and application. Catechetical structure.

Also add: "The plan file specifies which register this unit uses. If no register is specified, use narrative-historical as the default. The STYLE-GUIDE.md 'Sacred Registers' section describes how THIS WRITER handles each register -- always defer to STYLE-GUIDE.md over these generic descriptions."
