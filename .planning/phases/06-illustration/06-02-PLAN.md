    **character-ref.md:**
    Usage: /scr:character-ref <name> [--style <style>]

    Generate a character visual reference sheet prompt. Load:
    - .manuscript/config.json for work_type
    - data/CONSTRAINTS.json for availability and adaptations (sacred: figure-ref)
    - CHARACTERS.md (or FIGURES.md for sacred, via file_adaptations)
    - ART-DIRECTION.md if it exists (for style consistency)

    Find character by name (case-insensitive, same pattern as character-sheet.md). Extract physical description, clothing, distinguishing features, emotional range.

    Generate structured prompt per D-01 with sections:
    - Subject: Full character description (height, build, coloring, distinguishing features)
    - Poses: Front view, 3/4 view, profile view, action pose
    - Expressions: Key emotional states from character profile (e.g., determined, grieving, joyful)
    - Clothing: Default outfit, alternate outfits if mentioned in profile
    - Props/Accessories: Items associated with the character
    - Style: Match ART-DIRECTION.md style if exists, otherwise derive from genre
    - Technical Specs: Recommend dimensions (character sheet typically 2400x3200px at 300 DPI)

    Save to .manuscript/illustrations/character-refs/{character-name}-ref.md

    **chapter-header.md:**
    Usage: /scr:chapter-header [--style <style>] [--chapter <ref>]

    Generate decorative chapter header/ornament design prompts. Load:
    - .manuscript/config.json for work_type, genre
    - data/CONSTRAINTS.json for availability (prose and sacred only)
    - ART-DIRECTION.md if exists
    - OUTLINE.md if --chapter specified (to get chapter title/theme)

    Style options: floral, geometric, art-deco, gothic, minimalist, ornate, hand-drawn, woodcut, custom
    Default style: derive from genre (fantasy->ornate, literary->minimalist, sacred->geometric/arabesque, etc.)

    Generate prompt per D-01 with sections:
    - Subject: Ornamental design description (motifs, symbols relevant to chapter/book theme)
    - Composition: Horizontal banner format, centered, with space for chapter number/title
    - Style: Matching selected or derived style
    - Color Palette: Match ART-DIRECTION.md or derive (many chapter headers are single-color/monochrome)
    - Technical Specs: Width matches page text block width, height ~1-2 inches, 300 DPI, transparent background recommended

    If --chapter specified, incorporate chapter-specific thematic elements (a sword for a battle chapter, a rose for a romance scene, etc.)

    Save to .manuscript/illustrations/chapter-headers/chapter-header-prompt.md (or chapter-{ref}-header-prompt.md if specific chapter)
  </action>

    Create commands/scr/map-illustration.md following established command pattern.

    Usage: /scr:map-illustration [--region <area>] [--style <style>]

    Generate world/location map illustration prompts from WORLD.md geographic content. Load:
    - .manuscript/config.json for work_type
    - data/CONSTRAINTS.json for availability and prerequisites (requires WORLD.md)
    - WORLD.md (or COSMOLOGY.md for sacred, CONTEXT.md for academic via file_adaptations)
    - ART-DIRECTION.md if exists

    If WORLD.md missing or has no geographic content, guide: "Run `/scr:build-world --area geography` first to establish your world's geography."

    Extract from WORLD.md:
    - Geographic features (continents, oceans, mountains, rivers, forests)
    - Key locations (cities, settlements, landmarks)
    - Political boundaries if mentioned
    - Climate/biome information
    - Travel routes or important paths

    If --region specified, focus on that area. Otherwise generate full world map prompt.

    Style options: fantasy-parchment, satellite-realistic, hand-drawn, watercolor, schematic, antique, topographic
    Default: fantasy-parchment for fantasy/sacred, hand-drawn for literary, schematic for sci-fi

    Generate prompt per D-01 with sections:
    - Subject: Map of [world/region name] showing [key geographic features]
    - Composition: Top-down cartographic view, compass rose placement, legend/key area, title cartouche
    - Style: Selected or genre-derived style
    - Color Palette: Terrain-coded (blue=water, green=forest, brown=mountains, tan=desert) matching ART-DIRECTION.md if exists
    - Mood: Adventure, scholarly, ancient, mystical -- derived from genre/tone
    - Labels: List of place names to include (extracted from WORLD.md)
    - Technical Specs: Full page width (typically 6"x9" or matching trim size), 300 DPI

    Save to .manuscript/illustrations/maps/world-map-prompt.md (or region-{name}-map-prompt.md)
